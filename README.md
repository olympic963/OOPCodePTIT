Trên mạng có thể sẽ có rất nhiều repo up source code trên PTIT code rồi, mình up lại để chỉnh 1 số bài vào ra file thường xuyên bị TLE mặc dù thuật toán rất tốt. 
Đối với các bài vào ra file, các bạn sử dụng các class dùng để đọc hoặc chuyển đổi input có chứa tiền tố Buffered như BufferredReader hoặc BufferedInputStream chẳng hạn, vì các class này đọc dữ liệu theo KHỐI (block), không đọc từng ký tự nên chúng làm giảm số lần truy cập I/O hệ điều hành -> xử lý nhanh hơn. Thông thường với các bài đọc file Object, kích thuớc input rất lớn, thuật toán ngon thì thường chạy khoảng 2,5s cho FileInputStream, tuy nhiên với BufferedInputStream thì thời gian này chỉ khoảng 0,5s.
